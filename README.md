# 💳 Smart ATM System (Java)

## 📌 Project Overview
This project is a **Smart ATM Simulation** built using Java.  
It mimics real-world ATM operations like withdrawal, deposit, and balance tracking using ₹100 and ₹500 notes.

---

## 🚀 Features
- 💸 Withdraw cash (₹100 & ₹500 notes only)
- 💰 Deposit notes into ATM
- 📊 Check ATM status (available cash)
- 🧾 Mini statement (last transaction)
- ⚠️ Input validation (multiples of 100, withdrawal limit)

---

## 🛠️ Technologies Used
- Java
- Scanner (for user input)
- Basic control structures (loops, conditionals)
